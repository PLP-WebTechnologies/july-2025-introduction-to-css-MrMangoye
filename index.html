Understanding the Box Model

Every element in web design is a rectangular box. The CSS box model describes how these boxes are laid out on the page.

The box model consists of: content, padding, border, and margin.

Margin

The outer layer that creates space between elements.

Border

The edge surrounding the padding and content.

Padding

The space between content and border.
